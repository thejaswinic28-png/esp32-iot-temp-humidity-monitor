# ESP32 IoT Temperature and Humidity Monitor

## Overview
This project is a cloud-connected IoT monitoring system built using ESP32 and DHT22 sensor. The system reads real-time temperature and humidity data and uploads it to ThingSpeak cloud for live visualization.

---

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- ESP32 simulation using Wokwi
- ThingSpeak cloud integration
- Serial monitor output
- Virtual IoT prototyping

---

## Technologies Used
- ESP32
- DHT22 Sensor
- Wokwi Simulator
- ThingSpeak
- Arduino C++
- IoT Cloud Integration
